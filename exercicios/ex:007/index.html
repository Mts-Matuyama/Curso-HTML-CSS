Principais Formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <B> (não semântica)

Nesta frase, tenho um termo em destaque usando a tag <Strong> (semântico).

Itálico / Enfase

Nesta frase, temos um termo em itálico usando a tag <I> (não semântica)/p>

Nesta frase, temos um termo em enfase usando a tag <EM> (semântica)

Texto marcado

Para criar uma marcação no texto, utilizamos a tag <Mark>

Ao alterar a cor na marcação, deve se fazer no css, para que fique todos alterados igual foi feito aqui, caso contrário terá que colorir de uma em uma

Texto grande e pequeno

É possível criar textos ou palavras grandes e também pequeno, utilizando as tag <big> e <small>

Texto deletado

É possível marcar um texto como excluido, para indicar que ele deve ser lido, mas não considerado utilizando a tag <del> (semântica)

Texto inserido

Também podemos marcar um texto como inserido para dar uma enfase e indicar que ele foi adicionado depois utilizando a tag <ins> (semântica)

Existe também o sublinhado com a tag <U> (não semântica)

Texto sobrescrito

Para inserir coisas do tipo x20+3 utilizanto a tag <sup>

Texto subscrito

Para inserir coisas do tipo H2O utilizando a tag <sub>
