Módulo 1, capítulo 7, assunto referente à HTML.

<h1> Sínaliza o primeiro título (principal), não é proíbido um síte ter mais de um.
<h2> Sinaliza um sub-titulo dentro do primeiro <h1>.
<h3> Sinaliza um sub-titulo localizado dentro de <h2>
<h4> Sinaliza um sub-titulo localizado dentro de <h3>
<h5> Sinaliza um sub-titulo localizado dentro de <h4>
<h6> Sinaliza um sub-titulo localizado dentro de <h5>, é o último subtítulo que existe em HTML

Título principal

Sub-título

Sub-titulo dentro do h2

Sub-titulo dentro do h3

Sub-titulo dentro do h4
Sub-titulo dentro do h5
